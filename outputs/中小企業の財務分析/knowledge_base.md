## 1. 財務分析の基本体系

### 4軸フレームワーク（中小企業診断士標準）

```
┌─────────────────────────────────────────────────┐
│              中小企業財務分析の4軸                 │
├──────────┬──────────┬──────────┬─────────────┤
│ 収益性    │ 安全性    │ 効率性    │  成長性      │
│Profitability│ Safety  │Efficiency│   Growth    │
│          │          │          │              │
│儲かって   │潰れない   │資産を     │  伸びている  │
│いるか     │か        │効率よく   │  か          │
│          │          │使えているか│             │
└──────────┴──────────┴──────────┴─────────────┘
```

### 財務諸表3表の関係

- **損益計算書（P/L）**: 収益性・成長性の分析基盤
- **貸借対照表（B/S）**: 安全性・効率性の分析基盤  
- **キャッシュフロー計算書（C/F）**: 実際の資金繰り把握

### Dupont ROA分解

```
ROA = 売上高経常利益率 × 総資本回転率

      経常利益           経常利益     売上高
  ───────────── = ────────── × ──────────
      総資本              売上高      総資本

【収益性の問題】→ 売上高経常利益率の改善
【効率性の問題】→ 総資本回転率の改善
```

## 2. 収益性分析

### 主要指標と計算式

| 指標 | 計算式 | 優良 | 平均 | 要注意 |
|------|--------|------|------|--------|
| 売上高総利益率 | 売上総利益÷売上高×100 | 25%↑ | 15-25% | 15%↓ |
| 売上高営業利益率 | 営業利益÷売上高×100 | 5%↑ | 2-5% | 2%↓ |
| 売上高経常利益率 | 経常利益÷売上高×100 | 5%↑ | 2-5% | 2%↓ |
| 売上高純利益率 | 当期純利益÷売上高×100 | 3%↑ | 1-3% | 1%↓ |
| ROA | 経常利益÷総資産×100 | 5%↑ | 2-5% | 2%↓ |
| ROE | 当期純利益÷自己資本×100 | 10%↑ | 5-10% | 5%↓ |
| EBITDAマージン | EBITDA÷売上高×100 | 8%↑ | 3-8% | 3%↓ |

### EBITDA計算式

```
EBITDA = 経常利益 + 支払利息 + 減価償却費
       = 営業利益 + 減価償却費
```

### 収益性改善の切り口（コンサルティング）

1. **売上総利益率改善**: 価格戦略見直し、原価低減（調達コスト削減、歩留まり改善）
2. **営業利益率改善**: 販管費削減（固定費見直し、人件費効率化）
3. **経常利益率改善**: 金融コスト削減（金利交渉、借入構成最適化）

### 交叉比率（Cross Ratio）
```
交叉比率 = 売上総利益率 × 商品回転率
         = 売上総利益 ÷ 平均在庫金額

→ 商品ごとの収益効率を評価。低利益率でも高回転商品は有効。
```

## 3. 安全性分析

### 短期安全性指標

| 指標 | 計算式 | 優良 | 要注意 |
|------|--------|------|--------|
| 流動比率 | 流動資産÷流動負債×100 | 150%↑ | 100%↓ |
| 当座比率 | 当座資産÷流動負債×100 | 100%↑ | 80%↓ |

```
当座資産 = 現金・預金 + 受取手形 + 売掛金 + 有価証券
（棚卸資産・前払費用は除く）
```

### 長期安全性指標

| 指標 | 計算式 | 優良 | 要注意 |
|------|--------|------|--------|
| 自己資本比率 | 自己資本÷総資本×100 | 40%↑ | 20%↓ |
| 負債比率 | 負債÷自己資本×100 | 100%↓ | 200%↑ |
| 固定比率 | 固定資産÷自己資本×100 | 100%↓ | 150%↑ |
| 固定長期適合率 | 固定資産÷（自己資本＋固定負債）×100 | 100%↓ | - |
| インタレストカバレッジ（ICR） | 事業利益÷支払利息 | 8倍↑ | 3倍↓ |
| 債務償還年数 | 有利子負債÷（経常利益＋減価償却費） | 5年↓ | 10年↑ |

### 事業利益（ICR計算用）
```
事業利益 = 営業利益 + 受取利息・配当金
```

### 資金ショートのメカニズム
- 売掛金の長期化 → 資金回収遅延
- 棚卸資産の滞留 → 資金固定化
- 急激な売上拡大 → 運転資金需要増加（過剰融資リスク）
- 買掛金の急減 → 支払早期化による流動性低下

### 運転資金の計算
```
正常運転資本 = 売上債権 + 棚卸資産 - 買入債務
必要運転資金 = 月商 × (売掛金回転月数 + 棚卸資産回転月数 - 買掛金回転月数)
```

## 4. 効率性分析・CCC

### キャッシュコンバージョンサイクル（CCC）

```
CCC = 売掛金回転日数 + 棚卸資産回転日数 − 買掛金回転日数

  売掛金回転日数 = 売掛金残高 ÷ (売上高÷365)
  棚卸資産回転日数 = 棚卸資産残高 ÷ (売上原価÷365)
  買掛金回転日数 = 買掛金残高 ÷ (仕入高÷365)

  CCCが小さい（短い）ほど資金効率が良い
```

### 主要効率性指標

| 指標 | 計算式 | 優良 | 要注意 |
|------|--------|------|--------|
| 総資産回転率 | 売上高÷総資産 | 1.5回↑ | 0.6回↓ |
| 固定資産回転率 | 売上高÷固定資産 | 2.0回↑ | 1.0回↓ |
| 売掛金回転日数 | 売掛金÷(売上高÷365) | 30日↓ | 60日↑ |
| 棚卸資産回転日数 | 棚卸資産÷(売上原価÷365) | 30日↓ | 60日↑ |
| 買掛金回転日数 | 買掛金÷(仕入高÷365) | - | - |

### CCC改善策

1. **売掛金回転日数の短縮**: 与信管理強化、回収期日管理、早期回収インセンティブ
2. **棚卸資産回転日数の短縮**: 在庫適正化、ABC分析、デッドストック削減
3. **買掛金回転日数の延長**: 支払条件見直し（仕入先との交渉）

### 事例数値（PDF収録）
```
製造業事例：
  固定資産回転率 = 47.7回
  売掛金回転日数 = 38.6日
  棚卸資産回転日数 = 26.9日
  買掛金回転日数 = 27.8日
  CCC = 38.6 + 26.9 - 27.8 = 37.7日
```

## 5. 生産性分析・付加価値

### 付加価値の計算（加算法）

```
付加価値額 = 経常利益 + 人件費 + 減価償却費 + 支払利息 + 賃借料

  ※ 加算法（中小企業庁方式）：各分配先への分配額を合計
  ※ 控除法（日銀方式）：売上高 - 外部購入費用（原材料費、外注費等）
```

### 生産性指標

| 指標 | 計算式 |
|------|--------|
| 労働生産性（1人当たり付加価値） | 付加価値額 ÷ 従業員数 |
| 1人当たり売上高 | 売上高 ÷ 従業員数 |
| 労働分配率 | 人件費 ÷ 付加価値額 × 100 |
| 付加価値率 | 付加価値額 ÷ 売上高 × 100 |
| 設備生産性 | 付加価値額 ÷ 有形固定資産 |
| 資本集約度 | 有形固定資産 ÷ 従業員数 |

### 業種別 労働生産性・労働分配率の目安

| 業種 | 1人当たり付加価値（万円） | 労働分配率 |
|------|--------------------------|-----------|
| 製造業 | 500 | 55% |
| 卸売業 | 400 | 65% |
| 小売業 | 380 | 70% |
| 建設業 | 550 | 60% |
| サービス業 | 450 | 62% |
| IT・情報通信業 | 700 | 55% |
| 飲食業 | 300 | 75% |
| 不動産業 | 800 | 45% |

### 労働生産性改善の切り口
1. **付加価値額増加**: 収益性改善（価格↑・コスト↓）
2. **従業員数適正化**: 業務自動化、アウトソーシング活用
3. **設備投資による効率化**: 省力化投資のROI評価

## 6. 成長性分析

### 成長性指標

| 指標 | 計算式 | 優良 | 要注意 |
|------|--------|------|--------|
| 売上高増加率 | (今期売上高-前期売上高)÷前期売上高×100 | 5%↑ | 0%↓ |
| 営業利益増加率 | (今期営業利益-前期営業利益)÷前期営業利益×100 | - | - |
| 総資産増加率 | (今期総資産-前期総資産)÷前期総資産×100 | - | - |
| 従業員数増加率 | (今期人数-前期人数)÷前期人数×100 | - | - |

### 持続可能成長率（SGR）
```
SGR = ROE × (1 - 配当性向)
    = ROE × 内部留保率

→ 外部資金調達なしに維持できる最大成長率
```

## 7. 損益分岐点（CVP）分析

### 基本公式

```
損益分岐点売上高 = 固定費 ÷ (1 - 変動費率)
               = 固定費 ÷ 限界利益率

  変動費率 = 変動費 ÷ 売上高
  限界利益率 = 1 - 変動費率
  限界利益 = 売上高 - 変動費
```

### 安全余裕率（MOS）

```
安全余裕率 = (売上高 - 損益分岐点売上高) ÷ 売上高 × 100

  → 20%以上: 安全
  → 10-20%: 要注意
  → 10%以下: 危険
```

### 目標利益達成売上高

```
目標売上高 = (固定費 + 目標利益) ÷ 限界利益率
```

### 実例（PDF収録事例）

```
事例企業データ:
  売上高: 1,322百万円
  固定費: 375百万円
  変動費率: 68.3%
  限界利益率: 31.7%

  損益分岐点売上高 = 375 ÷ 0.317 = 1,183百万円
  安全余裕率 = (1,322 - 1,183) ÷ 1,322 × 100 = 10.5%

  目標売上高（目標利益=100百万円の場合）:
  = (375 + 100) ÷ 0.317 = 1,498百万円 (+13.3%)
```

### 変動費・固定費の分類

**変動費（売上連動）**: 原材料費、商品仕入高、外注費、運搬費、販売手数料
**固定費（売上非連動）**: 人件費、地代家賃、減価償却費、支払利息、広告宣伝費（一部）

## 8. キャッシュフロー分析

### CF計算書の構造

```
【営業CF】: 本業からの資金流出入
  当期純利益
  + 減価償却費（非資金費用加算）
  ± 売掛金・棚卸資産の増減
  ± 買掛金の増減
  ─────────────
  営業CF計

【投資CF】: 設備投資・有価証券売買等
  - 有形固定資産の取得
  + 有形固定資産の売却
  ─────────────
  投資CF計

【フリーCF】= 営業CF + 投資CF

【財務CF】: 借入・返済・増資等
  + 借入金増加
  - 返済
  ± 増資・配当
  ─────────────
  財務CF計

ネットCF = 営業CF + 投資CF + 財務CF
```

### 事例L社のCF計算書（PDF収録）

```
営業CF:
  当期純利益       806万円
  減価償却費       231万円
  売掛金受取手形の増加  △95万円
  棚卸資産の減少     51万円
  その他流動資産減少   39万円
  支払手形・買掛金の減少 △373万円
  その他負債の増加    49万円
  計             708万円

投資CF:
  有価証券の売却     67万円
  土地の売却       282万円
  減価償却資産の増加  △248万円
  建設仮勘定の減少    33万円
  無形固定資産の増加   △5万円
  その他固定資産の増加 △128万円
  投資有価証券の購入  △233万円
  繰延資産の増加      △7万円
  計            △239万円

フリーCF: 469万円

財務CF:
  借入金の増加     △133万円
  社債の増加       217万円
  増資            30万円
  自己株式の取得   △141万円
  剰余金の配当     △50万円
  計            △77万円

```

### CF5パターン診断

| パターン | 営業CF | 投資CF | 財務CF | 診断 |
|---------|--------|--------|--------|------|
| 優良企業型 | ＋ | △ | △ | 本業好調、投資も積極的、借入返済も進む |
| 成長投資型 | ＋ | △ | ＋ | 本業好調、積極投資のため借入増 |
| 事業転換型 | ＋ | ＋ | △ | 資産売却して借入返済 |
| 危機対応型 | △ | ＋ | ＋ | 本業低迷、資産売却＋借入で対応 |
| 経営危機型 | △ | △ | ＋ | 本業低迷、投資も継続、借入依存 |

### 売上高営業CF比率
```
売上高営業CF比率 = 営業CF ÷ 売上高 × 100

  → 売上高営業利益率より高い: 売掛金早期回収・買掛金支払遅延等で資金効率良
  → 売上高営業利益率より低い: 利益は出ているが資金繰りは悪化（逆の状況）
```

### CFインタレストカバレッジレシオ
```
CFインタレストカバレッジ = (営業CF + 支払利息・割引料 + 税金) ÷ 支払利息・割引料

  事例L社: (708 + 365 + 679) ÷ 365 = 4.8倍
  → 金融機関が融資先の信用格付けで重視する指標
```

### 資金繰り改善策

1. **売掛金サイトの短縮**: 支払期日管理、早期回収割引の導入
2. **買掛金サイトの延長**: 仕入先との交渉、支払条件見直し
3. **在庫削減**: 適正在庫管理、発注点見直し
4. **設備投資計画の平準化**: 投資時期の分散、リース活用

## 9. 設備投資評価

### 評価手法一覧

#### (1) 回収期間法（Payback Period）

```
回収期間 = 設備投資額 ÷ 年平均フリーキャッシュフロー (年)

→ 短いほど良い
→ 欠点: 回収期間後の収入を考慮しない

例: 設備投資1,200万円
  A案: 年400万回収 → 回収期間 = 3年
  B案: 年300万回収 → 回収期間 = 4年（但し4-5年目も300万の収入あり）
  → 単純比較ではA案有利、但しB案は長期収益性で有利
```

#### (2) 平均投資利益率法（ARR）

```
平均投資利益率 = 年平均利益 ÷ 平均投資額 × 100

  平均投資額 = (初期投資額 + 残存価値) ÷ 2
→ 時間価値を考慮しない
```

#### (3) 正味現在価値法（NPV）

```
NPV = Σ [各期CF ÷ (1+割引率)^年数] - 初期投資額

  → NPV > 0: 投資価値あり
  → NPV が最大の案を選択
  → 割引率には WACC を使用
```

#### (4) 内部収益率法（IRR）

```
NPV = 0 となる割引率 = IRR

  → IRR > 資本コスト（WACC）であれば投資価値あり
  → 複数案の比較に有効
```

### リース活用の財務効果

- 初期投資額の圧縮 → 自己資本比率・流動比率の維持
- オフバランス効果（オペレーティングリース）
- 固定費化による損益分岐点の上昇に注意

## 12. 経営戦略と財務分析

### アンゾフのマトリクスと財務影響

| 戦略 | 内容 | 財務影響 |
|------|------|---------|
| 市場浸透 | 既存市場×既存製品 | 安定的・リスク低 |
| 市場開拓 | 新市場×既存製品 | 販管費増加リスク |
| 製品開発 | 既存市場×新製品 | 研究開発費増加リスク |
| 多角化 | 新市場×新製品 | 最高リスク・高投資 |

### 事業集中vs多角化の財務比較（製造業実績）

| 区分 | 企業数構成比 | 1企業当たり売上高 | 売上高営業利益率 | 売上高総利益率 | 売上高販売管理費比率 |
|------|------------|-----------------|----------------|--------------|-------------------|
| 専業（主業比率100%） | 46.3% | 18,733百万円 | 5.7% | 19.0% | 13.3% |
| 兼業25%未満 | 33.1% | 23,961百万円 | 4.3% | 17.1% | 12.9% |
| 兼業25〜50%未満 | 15.8% | 30,188百万円 | 4.7% | 20.8% | 16.0% |
| 兼業50%以上 | 4.8% | 28,226百万円 | 4.2% | 22.8% | 18.6% |
| 全体 | 100.0% | 22,725百万円 | 4.9% | 18.9% | 14.0% |

（出典: 経済産業省「企業活動基本調査」2022年度）

**解釈**: 専業は営業利益率が最高（5.7%）だが規模は小さい。多角化は規模拡大するが利益率が低下する傾向。

### 多角化戦略の財務チェックポイント

1. **既存経営資源の活用度**: 設備・人材・特許をどの程度活用できるか
2. **コスト変化**: 原材料費・仕入・販管費の節減または増加
3. **ライフサイクル対応**: 多角化によるリスク分散効果
4. **シナジー効果**: 生産・販売・技術面での相乗効果
5. **安全性への影響**: 設備投資・人材確保による財務リスク

### 新事業展開のリスク評価

```
リスク水準（低→高）:
  市場浸透 < 市場開拓 < 製品開発 < 多角化

新規性が高いほど成功時の先行者利益も大きい。
→ 自己資本比率など内部留保が高い企業は
  リスクの高い研究開発・海外進出を検討可能。
```

## 13. 企業ライフサイクルと財務指標

### ライフサイクルステージと特徴

#### 創業期（創業〜6年）

**外部経営資源の活用**:
- 設備投資: リース制度活用、外注工場の利用
- 人材確保: 人材派遣の活用、人材スカウト
- 特許: ライセンス契約による活用
- 研究開発: 一部外部委託、試験研究機関の活用
- アウトソーシング: IT活用など各種業務委託

**公的支援制度**:
- 日本政策金融公庫による再チャレンジ支援融資制度
- 商工組合中央金庫の融資制度
- 信用保証協会による創業関連保証制度、事業承継保証制度
- IT活用・テレワーク導入向け日本政策金融公庫特別貸付（IT活用促進資金）

#### 成長期（創業7〜17年）

**特徴**:
- 創業期を上回る積極的な設備投資
- 売上高の増大による収益性上昇（ホップからステップへ）
- 浮動客が多いため、景気後退時の顧客維持コストが高い

**売上高対販売管理費比率の推移（製造業）**:
| 年度 | 成長期 | 衰退期 |
|------|--------|--------|
| 平成30年 | 28.0% | 19.6% |
| 令和2年 | 31.3%（+3.3%） | 20.5%（+0.9%） |
| 令和4年 | 26.3%（△5.0%） | 19.3%（△1.2%） |

**売上高総利益率の推移**:
- 成長期: 31.0%→32.9%（+1.9%）→29.1%（△3.8%）
- 衰退期: 22.6%→22.7%（+0.1%）→22.3%（△0.4%）

#### 衰退期（業歴長期）

**特徴**:
- 固定客が多く、景気後退時の対応が柔軟
- 販管費の節減効果が成長期企業より大きい
- 景気変動への経験豊富な対応力

### ライフサイクルと成長戦略選択

```
成長期の成長加速戦略:
1) 顧客維持・拡大: 既存顧客のロイヤルティ向上
2) 競合対応: 自社製品と競合する製品への対策
3) 需要変化対応: 製品改良、新商品開発、価格引下げ
4) 媒体・広告内容の変更
```

### 新事業展開のシナジー効果

```
例: 介護用ベッドの開発
  車いすメーカーが開発 → 販売面のシナジー効果（相乗効果）が期待できる
  他に生産面・技術面のシナジーも考慮

事業のポートフォリオ効果:
  既存事業継続しながら新事業追加（多角化）
  → 新事業失敗でも既存事業の貢献が期待できる
  → 事業転換（既存事業廃止）は既存事業貢献なし = リスク大
```

## 15. RAGシグナル判定基準

### システム判定ロジック

```python
# GREEN: 優良水準
# AMBER: 要注意水準（業種平均前後）
# RED: 問題水準

def rag_signal(value, low, high, inverse=False):
    if inverse:  # 低いほど良い指標（負債比率等）
        if value <= low: return "GREEN"
        elif value <= high: return "AMBER"
        else: return "RED"
    else:  # 高いほど良い指標
        if value >= high: return "GREEN"
        elif value >= low: return "AMBER"
        else: return "RED"
```

### 指標別 RAG閾値

| 指標 | GREEN (優良) | AMBER (平均) | RED (要注意) | inverse |
|------|-------------|-------------|-------------|---------|
| 売上総利益率 | 25%↑ | 10-25% | 10%↓ | No |
| 営業利益率 | 5%↑ | 2-5% | 2%↓ | No |
| 経常利益率 | 5%↑ | 2-5% | 2%↓ | No |
| ROA | 5%↑ | 2-5% | 2%↓ | No |
| ROE | 10%↑ | 5-10% | 5%↓ | No |
| 流動比率 | 130%↑ | 100-130% | 100%↓ | No |
| 当座比率 | 100%↑ | 80-100% | 80%↓ | No |
| 自己資本比率 | 40%↑ | 20-40% | 20%↓ | No |
| 負債比率 | 100%↓ | 100-200% | 200%↑ | Yes |
| ICR | 8倍↑ | 3-8倍 | 3倍↓ | No |
| 純有利子負債/EBITDA | 2倍↓ | 2-4倍 | 4倍↑ | Yes |
| 総資産回転率 | 1.2回↑ | 0.6-1.2回 | 0.6回↓ | No |
| 売掛金回転日数 | 30日↓ | 30-60日 | 60日↑ | Yes |
| 棚卸資産回転日数 | 30日↓ | 30-60日 | 60日↑ | Yes |
| CCC | 20日↓ | 20-60日 | 60日↑ | Yes |
| 売上成長率 | 3%↑ | 0-3% | 0%↓ | No |
| EBITDAマージン | 8%↑ | 3-8% | 3%↓ | No |

### 総合評価ロジック

```python
# RED指標が3つ以上 → 総合判定: RED（要緊急対応）
# GREEN指標が5つ以上かつRED=0 → 総合判定: GREEN（優良）
# それ以外 → 総合判定: AMBER（要注意・改善余地あり）

if red_cnt >= 3:
    overall_rag = "RED"
elif green_cnt >= 5 and red_cnt == 0:
    overall_rag = "GREEN"
else:
    overall_rag = "AMBER"
```

## 16. 診断書・報告書の構成フレームワーク

### 10ページ Word報告書 構成例

```
表紙・エグゼクティブサマリー (1ページ)
  - 企業概要、診断目的
  - 総合判定（RAG）と最重要課題3点
  - 即効策・中期施策のハイライト

第1章: 収益性分析 (2ページ)
  - 売上・利益推移チャート
  - DuPont分解（ROA、収益性×回転率）
  - 業種ベンチマーク比較表
  - 問題点と改善提言

第2章: 安全性分析 (1.5ページ)
  - 流動比率・当座比率・自己資本比率
  - 有利子負債分析（借入依存度・返済能力）
  - 資金繰りリスクの評価

第3章: 効率性・CCC分析 (1.5ページ)
  - CCC分析（売掛・在庫・買掛の3軸）
  - 回転率の経年推移
  - 改善余地の定量試算

第4章: 生産性分析 (1ページ)
  - 付加価値計算
  - 1人当たり生産性指標
  - 労働分配率の適正評価

第5章: 損益分岐点分析 (1ページ)
  - 変動費・固定費の分類
  - BEP計算・安全余裕率
  - 目標利益達成のシナリオ

第6章: 戦略提言 (2ページ)
  - 短期施策（0-6ヶ月）: 即効性の高い改善
  - 中期施策（6-18ヶ月）: 体質改善
  - 長期施策（18ヶ月+）: 成長戦略
  - アクションプランと KPI
```

### コンサルティング提言フレームワーク（McKinsey/BCG式）

```
Issue Tree（論点分解）:
  ├── 収益性が低い
  │   ├── 粗利率の問題（価格・コスト）
  │   └── 費用構造の問題（固定費比率）
  ├── 安全性が低い
  │   ├── 流動性リスク（短期）
  │   └── 財務レバレッジリスク（長期）
  └── 成長性が低い
      ├── 既存事業の縮小
      └── 新規事業の不在

解決策の優先順位:
  QUICK WIN: 6ヶ月以内・高インパクト
  STRATEGIC: 12-24ヶ月・持続効果大
  INVEST: 中長期・高投資・高リターン
```
