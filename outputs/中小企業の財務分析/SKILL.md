# 中小企業 財務分析・コンサルティング自動生成スキル

## 概要

中小企業の財務データを入力すると、**中小企業診断士レベル × 大手コンサルティングファーム（McKinsey / BCG / Roland Berger 流）** の切り口で：

1. **Wordレポート（約10ページ）** — エグゼクティブサマリー、収益性・安全性・効率性・成長性分析、課題特定、戦略提言
2. **Excelモデル** — 財務3表モデル、経営指標ダッシュボード、シナリオ分析、業界比較チャート

を自動生成します。

---

## トリガー条件

以下のいずれかの発言があったら必ずこのスキルを使う：

- 「財務分析して」「財務診断」「財務レポートを作成して」
- 「中小企業診断士レベルで分析」「コンサルティング提案書」
- 「Wordレポートを作って」「Excelモデルを作って」
- 「PL/BS/CFを分析」「経営改善提案」「資金調達支援」
- 会社名・売上・利益などの財務数値が貼り付けられた場合

## 実行手順

### Step 1 — データ収集・確認

データが不足している場合は以下の優先度で補完する：
- **必須**: 売上高、営業利益、総資産、純資産
- **推奨**: 流動資産・流動負債（安全性分析用）
- **任意**: CF、従業員数、複数期データ

不足項目があればユーザーに確認する。最低限の項目がある場合はその範囲で分析を実施し、「データ制約」として明記する。

### Step 2 — 財務分析の実施（中小企業診断士フレームワーク）

以下の4軸 × 定量指標 × 定性評価で分析する：

#### ① 収益性分析
| 指標 | 計算式 | 業界平均目安 |
|------|--------|------------|
| 売上高総利益率 | 売上総利益 ÷ 売上高 | 業種別 |
| 売上高営業利益率 | 営業利益 ÷ 売上高 | 業種別 |
| 売上高経常利益率 | 経常利益 ÷ 売上高 | 業種別 |
| ROA（総資産利益率） | 経常利益 ÷ 総資産 | 3〜8% |
| ROE（自己資本利益率） | 当期純利益 ÷ 純資産 | 8〜15% |
| EBITDA マージン | EBITDA ÷ 売上高 | 業種別 |

#### ② 安全性分析
| 指標 | 計算式 | 安全基準 |
|------|--------|---------|
| 流動比率 | 流動資産 ÷ 流動負債 | 120%以上 |
| 当座比率 | (流動資産-棚卸) ÷ 流動負債 | 100%以上 |
| 自己資本比率 | 純資産 ÷ 総資産 | 40%以上 |
| 負債比率 | 有利子負債 ÷ 自己資本 | 100%以下 |
| インタレストカバレッジ | 営業利益 ÷ 支払利息 | 5倍以上 |
| ネット有利子負債/EBITDA | (借入-現金) ÷ EBITDA | 3倍以下 |

#### ③ 効率性分析
| 指標 | 計算式 | 改善目安 |
|------|--------|---------|
| 総資産回転率 | 売上高 ÷ 総資産 | 業種別 |
| 売掛金回転日数 | 売掛金 ÷ (売上高/365) | 30〜60日 |
| 棚卸資産回転日数 | 棚卸 ÷ (売上原価/365) | 業種別 |
| 買掛金回転日数 | 買掛金 ÷ (売上原価/365) | 業種別 |
| CCC（現金転換サイクル） | 売掛日数 + 在庫日数 - 買掛日数 | 短いほど良 |

#### ④ 成長性分析
| 指標 | 計算式 |
|------|--------|
| 売上高成長率 | (当期-前期) ÷ 前期 |
| 営業利益成長率 | (当期-前期) ÷ 前期 |
| 自己資本成長率 | (当期-前期) ÷ 前期 |

### Step 3 — 課題の特定（コンサルティングフレームワーク）

**MECE原則**で課題を整理：

1. **収益構造課題** — マージン低下、コスト高、売価政策
2. **財務健全性課題** — 過剰債務、運転資本逼迫、資本効率
3. **事業効率課題** — 在庫過多、回収遅延、固定費肥大化
4. **成長・競争力課題** — 売上停滞、市場縮小、差別化不足

各課題は **「現状 → 原因仮説 → 影響」** の構造で記述する。

### Step 4 — 戦略提言（大手コンサル流）

優先度マトリクス（緊急度 × 重要度）で施策を分類：

- **即時対応（0〜3ヶ月）**: キャッシュフロー改善、コスト削減
- **短期施策（3〜12ヶ月）**: 収益構造改革、営業力強化
- **中長期施策（1〜3年）**: 事業ポートフォリオ最適化、新規事業

各提言は **「施策 → KPI → 期待効果（定量）→ 実行体制」** で記述。

### Step 5 — 成果物生成

以下のスクリプトを実行して成果物を生成：

```bash
cd /path/to/outputs
python3 generate_sme_report.py --data input_data.json
```

または Python から直接呼び出す：

```python
from generate_sme_report import SMEFinancialAnalyzer
analyzer = SMEFinancialAnalyzer(data)
analyzer.generate_word_report("output_report.docx")
analyzer.generate_excel_model("output_model.xlsx")
```

## 成果物の構成

### Word レポート（10ページ構成）

| ページ | 内容 |
|--------|------|
| P1 | 表紙（会社名・診断日・診断機関） |
| P2 | エグゼクティブサマリー（信号機評価＋Top3課題＋提言） |
| P3-4 | 財務ハイライト（KPIカード＋主要指標テーブル） |
| P5 | 収益性分析（チャート＋解説） |
| P6 | 安全性・効率性分析 |
| P7 | 成長性・キャッシュフロー分析 |
| P8 | 課題の特定と根本原因分析 |
| P9-10 | 戦略提言ロードマップ＋アクションプラン |

### Excel モデル

| シート | 内容 |
|--------|------|
| ① 財務3表 | PL・BS・CF の入力＋自動集計 |
| ② 経営指標 | 全KPI自動計算＋信号機表示（RAG） |
| ③ 趨勢分析 | 複数期折れ線チャート（最大5期） |
| ④ 業界比較 | 業種別平均との比較棒グラフ |
| ⑤ シナリオ | 楽観・中立・悲観の3シナリオP/L |
| ⑥ 資金繰り | 12ヶ月キャッシュフロー予測 |

## 品質基準

- **中小企業診断士の診断基準** に準拠（中小企業庁の財務指標DB参照）
- **業種別平均値** を比較対象として明示（製造業・卸売業・小売業・サービス業・建設業）
- **根拠のある提言** — データから導いた仮説を明示し、推測と事実を区別
- **実行可能性** — 中小企業の人的・資金的リソースを考慮した現実的な施策のみ提言

## 注意事項

- 本スキルの出力は **経営判断の参考情報** であり、公認会計士・税理士による正式な財務諸表監査の代替ではない
- 業界平均値は中小企業庁「中小企業実態基本調査」および TKC 経営指標（BAST）を参考に設定
- データの入力誤りは分析精度に直結するため、入力値の確認を推奨する
